cm-chessboard

The (upcoming) chessboard for chessmail.de / chessmail.eu

A Lightweight ES6 module based responsive SVG chessboard with almost no external dependencies.

Repository and documentation on GitHub.

Examples

  • Simple chessboards, view only
  • Responsive chessboard
  • Input enabled without validation
  • Input enabled, move validation with chess.js
  • Set different positions, slow, with animation
